# Cookiery.github.io

https://cookiery.github.io/

## Quick Start
```shell
brew install hugo
hugo server
```

## Write Blog
```shell
# create a draft
sh create-post.sh
```

## Deploy Blog
Auto deploy by github action, the config workflow is in the  `.github/workflows/hugo.yml`
